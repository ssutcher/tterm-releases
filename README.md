# tterm-releases

Public release artifacts for **Tink** — the internal TinkerBot terminal (a.k.a. `tterm` / T0).
macOS, Apple Silicon (arm64). Signed & notarized, so it opens without Gatekeeper warnings.

## Latest

**Tink 0.6.1** (built Jun 15, 2026)

[⬇ Download Tink_0.6.1.dmg](https://github.com/ssutcher/tterm-releases/releases/download/v0.6.1/Tink_0.6.1.dmg) · 143 MB

```bash
curl -L -O https://github.com/ssutcher/tterm-releases/releases/download/v0.6.1/Tink_0.6.1.dmg
```

## Install

1. Download the `.dmg` above.
2. Open it and drag **Tink.app** to `/Applications`.
3. Launch it — no right-click-to-open needed (notarized).
